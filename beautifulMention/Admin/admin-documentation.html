Screen 1 - Login

Route: /admin/login

The Login screen is the entry point to the AMS Admin Panel. It is powered by Filament 5's default authentication system and provides an email/password form protected by CSRF tokens. Upon successful credential verification, users are redirected to the admin dashboard. If the authenticated user has a configured app_authentication_secret, they are redirected to the MFA Challenge screen instead.

Screen 2 - Password Reset

Route: /admin/password-reset

The Password Reset screen allows users who have forgotten their credentials to request a password reset link via email. The system generates a time-limited token stored in the password_reset_tokens table and sends a reset link to the user's registered email address. Screen 3 - MFA Challenge

Route: /admin/login

The MFA (Multi-Factor Authentication) Challenge screen appears after successful email/password authentication for users who have a configured app_authentication_secret. The user must enter a 6-digit TOTP (Time-based One-Time Password) code from their authenticator app (e.g., Google Authenticator, Authy). The code is validated against the stored secret. Users can also use recovery codes stored in app_authentication_recovery_codes if they lose access to their authenticator device.

Screen 38 - Grade Export

Route: /admin/grade-export

The Grade Export screen enables coordinators to export student grades in CSV or PDF format. The page provides two dropdown filters: a Semester dropdown (scoped to the coordinator's assigned semesters, defaulting to the most recent active semester) and a Course dropdown (dynamically populated with courses that have projects in the selected semester).

After selecting filters, a preview table is rendered showing: university_id, student name, project title, calculated score, override score, final score (displayed with 2 decimal places), and letter grade (looked up from the grading scales table).

Export CSV: Streams a CSV download containing all rows visible in the preview table with the same columns.
Export PDF: Generates a PDF document using the Barryvdh DomPDF package, rendering the grade-report Blade view with formatted tables, headers, and institution branding.
